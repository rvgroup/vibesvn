# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-02-08

### Added
- **Advanced File Filtering**: Added powerful file filtering with wildcard support (*.dart, src/*, test?.js)
- **Enhanced UI Layout**: Redesigned commit screen interface with improved organization
- **New Application Icons**: Created custom app icons with modern gradient design
- **Namespace Migration**: Updated project namespace from com.example to com.rvgroup
- **Compact Interface**: Combined file filter and operation buttons into single efficient row

### Improved
- **User Experience**: Streamlined commit workflow with better visual hierarchy
- **File Management**: Enhanced file selection and filtering capabilities
- **Visual Design**: Modern gradient icons with consistent branding across platforms
- **Interface Organization**: Reorganized commit screen panels for better usability

### Fixed
- **Build Issues**: Resolved macOS icon generation problems
- **Duplicate Resources**: Fixed Android build conflicts with launcher icons
- **UI Layout**: Optimized spacing and alignment in commit interface

### Technical
- **Multi-platform Support**: Updated icons for macOS, Windows, iOS, Android, and Linux
- **Code Organization**: Improved commit screen structure and maintainability
- **Localization**: Added translations for new filtering features

---

## [1.0.0] - 2026-02-07

### Added
- **Initial Release**: Complete SVN client functionality
- **Repository Management**: Add, edit, and remove SVN repositories
- **File Status Tracking**: View and manage file changes with color-coded status indicators
- **Commit Operations**: Full commit workflow with message templates
- **Multi-language Support**: English and Russian localization
- **Cross-platform**: Support for macOS, Windows, Linux, iOS, and Android
- **Settings Management**: Comprehensive user preferences and configuration
- **Diff Integration**: Built-in and external diff tool support
- **History Tracking**: Commit history and message templates
